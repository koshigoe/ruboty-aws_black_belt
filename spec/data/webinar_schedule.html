AWS Black Belt Online Seminar シリーズ

AWS Black Belt Online Seminar シリーズは、製品・サービス別、ソリューション別、業種別のそれぞれのテーマに分かれて、アマゾン ウェブ サービス ジャパン株式会社が主催するオンラインセミナーシリーズです。

全国どこからでも、インターネット環境があればご参加いただけます。ご質問いただくこともできますので、会場まで足を運ぶのが難しい方もお気軽にご参加ください。

1/24（火） [AWS Black Belt Online Seminar] AWS 上の Jenkins 活用方法

時間： 12:00-13:00
対象： 技術者向け
セミナー概要：
DevOps スタイルの開発では自動化による継続的インテグレーション (CI) や継続的デプロイ (CD) の実現が欠かせません。Jenkins は広く使われているオープンソースのオートメーションツールです。このセミナーでは AWS 上で Jenkins を利用する際のベストプラクティスのほか、より効果的な利用方法としてコンテナ管理サービスであるECSとの連携、新しいビルドサービスである CodeBuild との連携についてお話しします。

時間： 18:00-19:00
対象： 技術者向け
セミナー概要：
AWS Storage Gateway は、オンプレミスのストレージ環境と AWS クラウド間でシームレスな階層型のハイブリッドストレージを実現するためのサービスです。このセミナーでは、新機能である File Gateway を含む AWS Storage Gateway の機能やユースケースをご紹介すると共に、他の AWS ストレージサービスとの使い分け等、AWS Storage Gateway の活用ポイントについてご説明します。

1/31（火） [AWS Black Belt Online Seminar] AWS で実現する Disaster Recovery

時間： 12:00-13:00
対象： 技術者向け
セミナー概要：
Disaster Recovery（災害対策）はシステムの重要な検討ポイントの 1 つですが、従来はコスト効果や構築及び復旧手順の複雑さなど多くの課題がありました。AWS を利用することで初期投資やランニングコストを抑制しつつ、容易に DR シナリオを実現することが可能になります。本セミナーでは DR に役立つサービスや機能、可用性レベルに応じた DR 構成例などを説明いたします。

対象： 技術者向け

AWS OpsWorks は Chef を使って、Amazon EC2 インスタンスまたはオンプレミスのコンピューティング環境でのサーバーの設定、デプロイ、および管理を自動化します。本セッションでは、新たに追加されたリージョンエンドポイント対応やAWS OpsWorks for Chef Automate, AWS CodePipeline との連携などの新機能、さらにデプロイメント戦略等の AWS OpsWorks スタックのベストプラクティスについて紹介いたします。

対象： 技術者向け
